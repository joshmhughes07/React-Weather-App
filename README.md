# React + Vite

This project uses React and Vite to run.

This is a simple but easy to use weather app that shows the next weeks weather (Temperature, Wind Speed and Rain)

Clicking on a day tile brings up expanded graphs built using react-chart-js

The Live Version is hosted on github pages here :https://joshmhughes07.github.io/React-Weather-App/



Dependencies:

React 
Vite 
npm
react-chart-js
